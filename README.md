# Medical Appointment Booking

## 📌 Project Overview
This is a frontend React-based Medical Appointment Booking application.  
It allows users to browse doctors, view details, search for specialties, and book appointments.

## 🚀 Features
- Navigation bar for easy routing between pages
- Doctor listing and doctor detail cards
- Search functionality to find doctors by specialty
- Appointment booking interface (UI)
- Responsive design for different screen sizes

## 🛠️ Technologies Used
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router DOM
